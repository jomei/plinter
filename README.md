# Неиспользуемая функция

Функция, которую никто не вызывает, или она вызвается в петле одиночных вызовов. 

## Примеры

### Неиспользуемая

Все 3 фунциии неиспользуемые

```
foo(){ bar() }

bar() { baz() }

baz() { foo() }
```

### Используемая

```
foo{
    baz();
    bar();
}

baz() {...}
bar() {...}
```
